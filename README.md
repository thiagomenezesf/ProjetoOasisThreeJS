# ProjetoOasisThreeJS
Este é um projeto de Computação Gráfica e trata-se de uma interface interativa que simula um ambiente semelhante ao de um Oásis, uma pequena região fértil em pleno deserto, graças à presença de água. Neste cenário, o usuário encontra um lago com algumas palmeiras rodeando o ambiente, além de uma barraca e uma fogueira para acampar. Na borda do lago, há uma canoa em cima de um conjunto de madeira. Perto da fogueira, temos um macaquinho.

O sistema foi projetado para web utilizando como biblioteca principal o Three.Js. Foram priorizados alguns aspectos de otimização, como redução de segmentos (faces) do solo, utilização de modelos de sombras mais leves, além de reduzir sua qualidade.

Para fazer o projeto funcionar basta utilizar o comando npx vite para criar um localhost que é capaz de rodar a interface. Se necessário, instalar o pacote npm utilizando o comando npm install.
